# HTTPMAN

A simple HTTP Client for testing REST API(s)

![screenshot](./screenshots/Screenshot1.png)

## Getting Started

### Dependencies

* Git
* Node

### Installing

* Clone this repo
* `cd` into the repo and run `yarn install` to install all the dependencies

### Executing program

If you have dependencies installed run this

```sh
yarn start
```
